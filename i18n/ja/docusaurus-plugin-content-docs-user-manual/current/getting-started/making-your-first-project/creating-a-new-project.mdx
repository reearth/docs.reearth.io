---
title: 新しいプロジェクトを作成する
サイドバー位置: 3
---

ログインが完了すると、最初に [ダウンロード](/user-manual/getting-started/understanding-reearth-ui/Dashboard) というページが表示されるようになります。

この画面では、新しいプロジェクトを簡単に作成することができます。

<Image path={require('./img/2-1.png').default} />

<Image path={require('./img/2-2.png').default} />

<Image path={require('./img/2-3.png').default} />

<Order>
  <ListByNumber>
    [新規プロジェクト]ボタンをクリックすると、[プロジェクトの初期化] という表示画面が開きます。
  </ListByNumber>
  <ListByNumber>
    プロジェクト名を記入
  </ListByNumber>
  <ListByNumber>
    プロジェクトの詳細を記入する
  </ListByNumber>
  <ListByNumber>
    サムネイル画像を選択・追加する
  </ListByNumber>
  <ListByNumber>
    新規プロジェクトの作成確認
  </ListByNumber>
  <ListByNumber>
    完了すると、プロジェクトリストに新しいプロジェクトの名前が表示されます。
  </ListByNumber>
</Order>

これで、最初のRe:Earthプロジェクトが作成されました。ここでプロジェクトカードの編集ボタンをクリックすると、先ほど作成したプロジェクトが表示されます。
