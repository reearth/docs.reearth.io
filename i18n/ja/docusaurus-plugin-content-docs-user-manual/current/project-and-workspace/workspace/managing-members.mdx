ワークスペースの設定ページで、ワークスペースのメンバーの管理をすることができます。

<Image path={require('./img/4-1.png').default} />

<Order>
  <List>
    <strong>新規メンバー</strong> ボタンで、メンバーの追加を行います。
  </List>
  <List>
    <strong>メンバーの権限</strong> ボタンでは、共有中のメンバーの権限を確認することができます。
  </List>
  <List>
    <strong>権限変更</strong> ボタンでは、ワークスペース内でのメンバーの権限を変更することができます。
  </List>
  <List>
    <strong>メンバーの削除</strong> ボタンをクリックすると、メンバーをワークスペースから削除することができます。
  </List>
</Order>

## メンバーの招待

以下の手順でメンバーをワークスペースに招待することができます。

<Image path={require('./img/4-2.png').default} width="70%" noRoundCorner/>

<Order>
  <ListByNumber>
    <strong>新規メンバー</strong>をクリックすると、<strong>メンバーを追加</strong> モーダルが表示されます。
  </ListByNumber>
  <ListByNumber>
    招待したいユーザーのユーザー名もしくはメールアドレスを入力します。
  </ListByNumber>
  <ListByNumber>
    招待したいユーザーが正しく表示されているか確認します。
  </ListByNumber>
  <ListByNumber>
    ユーザーが正しことを確認したら<strong>追加</strong> ボタンをクリックします。
  </ListByNumber>
  <ListByNumber>
    メンバーリストにメンバーが追加されます。
  </ListByNumber>
</Order>

:::info
チームワークスペースに追加できるのは、Re:Earthのアカウントを持っているユーザーだけです。
:::

:::caution
ユーザーネームは大文字と小文字を区別しますので、正確に入力してください。
:::

## 権限の種類

メンバーの権限は以下の３種類存在します。

- **オーナー**: ワークスペースの最高権限を持ち、ワークスペースに関する全ての設定を操作することができます。
- **編集**: ワークスペース内のプロジェクトの閲覧・編集・公開をすることができ、ワークスペース内にメンバーを追加することもできます。
- **閲覧**: 全てのプロジェクトを閲覧することができますが、編集及び設定を変更することはできません。

:::info
権限の範囲は現在検討中です。権限範囲の拡張が必要な場合は、開発をお待ちいただくかDiscordなどでお気軽にお問合せください。
:::

## メンバーの権限を変更する

各メンバーに対して、特定の操作を制限するためにそれぞれに権限が与えることができます。

メンバーの権限を変更するには次の操作を行います:

<Image path={require('./img/4-3.png').default} width="70%" noRoundCorner/>

<Order>
  <ListByNumber>
    メンバーリストの ✏️ ボタンをクリックします.
  </ListByNumber>
  <ListByNumber>
    ドロップダウンリストから、付与する権限を選択します。
  </ListByNumber>
  <ListByNumber>
    ☑️ ボタンをクリックして確定します。
  </ListByNumber>
</Order>

:::caution
権限の変更ボタンが表示されていない場合、自分の現在の権限では編集できないことを示しています。自身の権限を変更してもらうためにワークスペースのオーナーに連絡してください。
:::
