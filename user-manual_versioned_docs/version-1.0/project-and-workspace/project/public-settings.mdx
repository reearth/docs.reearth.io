Public settings allow you to set parameters that affect the published project.

<Image path={require('./img/4-1.png').default} />

<Order>
  <List>
    <strong>Public status</strong> allows you to check the current public status of your project.
  </List>
  <List>
    <strong>Basic Authorization</strong> supports adding basic user and password login to your project. Please check [here] for more details.
  </List>
  <List>
    <strong>Public Info</strong> is the basic project information that will be seen publicly(OGP included). Please check [here] for more details.
  </List>
  <List>
    <strong>Site settings</strong> allow you to change the domain of the project. The sub-domain of Re:Earth is used by default, or you can use a custom domain. Please check [here] for more details.
  </List>
</Order>
