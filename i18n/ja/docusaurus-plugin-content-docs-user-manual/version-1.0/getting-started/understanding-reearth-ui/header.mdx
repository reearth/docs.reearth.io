この機能は、プロジェクトの公開など、いくつかの機能を簡単に実行できるようにするためのものです。この機能は、RE:Earthのすべてのページで利用できます。ただ、この機能はページによって微妙に違いがあります。ここでは、編集者ページの見出しにある機能を例にとって説明します。

<Image path={require('./img/2-1.png').default} />

<Order>
  <List>
    ボタンでダッシュボードに移動
  </List>
  <List>
    ボタンでエディターへ移動
  </List>
  <List>
    ユーザーのサムネイル（開発中...）
  </List>
  <List>
    ユーザー名
  </List>
  <List>
    現在のワークスペース名
  </List>
  <List>
    アカウントとワークスペースの選択（詳細は下記をご覧ください）
  </List>
  <List>
    編集するプロジェクト名
  </List>
  <List>
    プロジェクトの情報一覧（詳細は下記をご覧ください。）
  </List>
  <List>
    プロジェクトプレビュー・ボタン
  </List>
  <List>
    プロジェクト公開ボタン
  </List>
</Order>

## アカウントとワークスペースの選択項目

<Image path={require('./img/2-2.png').default} width="50%"/>

<Order>
  <List>
    アカウント設定画面へ移動します
  </List>
  <List>
   Switch worspaceは、ワークスペースの切り替えを可能にします
  </List>
  <List>
    現在表示されているワークスペースはこちらです
  </List>
  <List>
   [ワークスペースの管理] をクリックすると、ワークスペースの概要ページが表示されます 
  </List>
  <List>
    ワークスペースの新規作成ボタンで、いつでも新しいワークスペースを作成可能です
  </List>
  <List>
    現在のアカウントからログアウトする
  </List>
</Order>

## プロジェクト機能の選択リスト

<Image path={require('./img/2-3.png').default} width="50%"/>

<Order>
  <List>
   このボタンをクリックすると、現在のプロジェクトの設定ページに移動します
  </List>
  <List>
   [データセット] ボタンをクリックすると、現在のプロジェクトが所有するすべてのデータの管理ページが表示されます。
  </List>
  <List>
   [プロジェクトの管理]ボタンを押すと、現在のワークスペースにあるすべてのプロジェクトが表示されます。
  </List>
  <List>
    このウェブサイトから資料作成ボタンがリンクされます。
  </List>
</Order>

## プレビューと公開

ヘッダーの右端には、プロジェクトのプレビューと公開に関する機能ボタンがあります。詳しい使い方は、[こちら]( /user-manual/1.0/getting-started/making-your-first-project/publishing-project)をご覧ください。 さらに詳細な情報については、[公開]( /user-manual/1.0/publication/overview)をご覧ください。
