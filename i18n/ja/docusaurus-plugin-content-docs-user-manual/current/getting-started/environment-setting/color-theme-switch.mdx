Re:Earthシステムでは、**Light theme**と**Dark theme**という2つのUIカラーテーマを提供しており、用途に応じていつでも切り替えることができます。

UIカラーテーマの切り替え機能は、アカウント設定画面でも利用可能です。
- ダッシュボードのヘッダー→アカウント設定
