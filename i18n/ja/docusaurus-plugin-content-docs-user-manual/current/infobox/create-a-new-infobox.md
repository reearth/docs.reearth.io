---
title: インフォボックスの作成
sidebar_position: 2
---

インフォボックスを作成するには、右パネルの上部のタブの中央の`インフォボックス`に切り替えます。

`インフォボックス作成`をクリックすると、選択しているマーカーにインフォボックスが追加されます。

![](./img/2_001.gif)

表示されたインフォボックスの`＋`から、インフォボックスに埋め込むコンテンツを選ぶことができます。　

![](./img/2_002.png)

コンテンツ形式は以下の5つから選択することができます。

- Text：テキスト
- Image：画像
- Video：動画
- Location：地図
- Table：表
