Re:Earthではワークスペーしにメンバーを招待することで、チームでプロジェクトを編集することができます。

## ワークスペースの作成

### ダッシュボードから

ダッシュボードの`クイックスタート` —> `新規ワークスペース作成` ボタン

<Image path={require('./img/2-1.png').default} />

### ヘッダーメニューから

ヘッダーのユーザーメニュー —> `ワークスペース` —> `新規ワークスペース`

<Image path={require('./img/2-2.png').default} />

### 設定画面から

<Image path={require('./img/2-3.png').default} />

:::tip
新しいワークスペースを作成すると、自動で作成したワークスペースに切り替わり、メンバーの追加やプロジェクトの作成をすぐに行うことができます。
:::
