---
title: 新しいアカウントを作成
sidebar_position: 2
---

Re:Earthは現在、内部開発の段階である為、一般登録アカウントに対応しておりません。そのため、[こちら](https://docs.google.com/forms/d/e/1FAIpQLSftlA7HKfSsCHND14jERCLh3YzDETj0tU2rPVHM8McQfPHt-g/viewform)からご連絡いただければ、新規にアカウントを開設いたします。

アカウントを取得し、認証が完了したら、[https://app.reearth.io/](https://app.reearth.io/)を開き、アカウント情報を入力してログインしてください。
<Image path={require('./img/1-1.png').default} />

通常、これでRe:Earthを使い始めることができます。もし、エラーが発生したり、ログインできない場合は、[こちら](https://docs.google.com/forms/d/e/1FAIpQLSftlA7HKfSsCHND14jERCLh3YzDETj0tU2rPVHM8McQfPHt-g/formResponse)までご連絡いただければ、トラブルシューティングのお手伝いをさせていただきます。
