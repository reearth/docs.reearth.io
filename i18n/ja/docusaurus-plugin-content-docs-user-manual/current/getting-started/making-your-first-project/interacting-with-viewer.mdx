## View Control

マウスを地球上にドラッグすると、視点を操作して今の地球を見られることが分かります。Re:Earthはデフォルトエンジンに[Cesium](https://cesium.com/)を使用しているため、視点の操作はCesiumを踏襲しています。

- マウス操作の場合。
  - **パンビュー** : 左クリック＋ドラッグ
  - **ズーム表示** : 右クリック＋ドラッグ、またはマウスホイールスクロール
  - **回転表示** : 中クリック＋ドラッグ、または CTRL＋左右クリック＋ドラッグ

- タッチパネルの場合。
  - **パンビュー**: 一本指でドラッグ
  - **ズーム表示** : 二本指でつまむ
  - **チルト表示** : 二本指でドラッグ、同方向
  - **回転表示** : 2 本指で反対方向にドラッグ

ぜひ、目の前の地球を自由に探索して、自分が今いる場所を探してみてください。
