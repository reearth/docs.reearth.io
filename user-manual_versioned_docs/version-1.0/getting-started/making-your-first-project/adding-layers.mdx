In Re:Earth, we can add layers to the map. And also objects that can be rendered on the digital earth (including [marker]( /user-manual/1.0/property/overview#marker-properties-1), [3D modal]( /user-manual/1.0/three-d-data/3d-model), [PhotoOverlay]( /user-manual/1.0/property/overview#photo-overlay-properties)...) are all managed as layers.

Here we will add a marker as layer to the map as an example.

## Changing layer properties

By selecting a layer, we can modify various properties of that layer to change its state or shape, such as the following examples.

### Changing position

<Image path={require('./img/4-3.gif').default} />

After pressing and holding the layer for more than 1 seconds, we can move its position on the map.

### changing apperance

<Image path={require('./img/4-4.gif').default} />

We can modify the style, size, color, etc. of the marker layer.

### Adding label

<Image path={require('./img/4-5.gif').default} />

We can also add a label to the marker layer, and adjust the font style, etc.

For the marker layer we have many customizable parts, please click [here]( /user-manual/1.0/property/overview#marker-properties) for more details. 
Different types of layers have their own different parameters and functions, you can find more information about them [here]( /user-manual/1.0/property/overview).

Please try to change the parameters of other layers, you may find something different.
