在Re:Earth中，我们可以给地图添加[layers]。所有可以在地图表面上呈现出来物体(包括[marker]，[3D modal]，[PhotoOverlay]...)都以layer的方式存在并管理。

这里我们以给地图添加marker作为layer为例子来说明。

## Adding a marker layer

[此处应该有步骤图]

1. 从[Tool bar]选中marker
2. 拖动Marker到地图上你想要放置的地方
3. 在[Outline]的layers list中便新建出一个叫marker的layer。

## Change layer properties

选中layer，我们便可以修改该layer的各种properties，来改变它的状态或者外形。

- 调整位置

[此处应该有牛逼的动图]

在layer上长按超过1s之后，我们便可以在地图上移动其位置。

- 调整呈现风格

[此处应该有牛逼的动图]

我们可以修改这个marker layer的式样，大小，颜色等等。

- 添加小标签

[此处应该有牛逼的动图]

我们还可以给marker layer添加小标签，以及调整字体的式样等等。

对于marker layer我们还有很多可以自定义的部分，详细介绍请查看[这里]。而不同类型的layer又有其各自不同的参数功能，[这里]你可以找到更多的介绍。
