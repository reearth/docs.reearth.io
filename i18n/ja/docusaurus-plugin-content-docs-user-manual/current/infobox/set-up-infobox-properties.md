インフォボックスの設定でできることは、以下の通りです。

- タイトル ：インフォボックス内のタイトルを決めることができます。

![](./img/3_001.png)

- サイズ：インフォボックスの大きさを変えることができます。

サイズ小
![](./img/3_002.png)
サイズ大
![](./img/3_003.png)

- 背景色：インフォボックスの色を決めることができます。

![](./img/3_004.gif)

- フォント:フォントサイズ、書体、色を決めることができます。

![](./img/3_005.png)

- フォントのレイアウト：左　中央　右　両端行揃え・太字・斜体・下線の設定ができます。
![](./img/3_006.gif)
