Re:Earth のプロジェクトは、とても簡単に作成することができます。

## 新規プロジェクトを作成する 2 つの方法

### ダッシュボードから

ダッシュボード —> クイックスタート —> `新規プロジェクト作成`ボタン —> [モーダルに情報を記入](/user-manual/project-and-workspace/project/create-a-new-project#新規プロジェクト作成モーダル)

<Image path={require("./img/2-1.png").default} />

### 編集画面から

編集画面 —> ヘッダーメニュー —> `プロジェクト設定` —> プロジェクト一覧ページ —> `新規プロジェクト作成`ボタン —> [モーダルに情報を記入](/user-manual/project-and-workspace/project/create-a-new-project#新規プロジェクト作成モーダル)

<Image path={require("./img/2-2.png").default} />

<Image path={require("./img/2-3.png").default} />

## 新規プロジェクト作成モーダル

最後に、以下のように表示されているモーダルにプロジェクトの初期情報を入力すれば、新規プロジェクトの作成は完了です。

<Image path={require("./img/2-4.png").default} />

<Order>
  <List>
    <strong>プロジェクト名</strong> プロジェクトの名前を記入する場所です。
    <em>(必須項目)</em>
  </List>
  <List>
    <strong>プロジェクト概要</strong>{" "}
    プロジェクトの概要や説明を記入する場所です。
  </List>
  <List>
    <strong>サムネイル画像</strong>{" "}
    プロジェクトのメインビジュアルを設定する場所です。
  </List>
  <List>上記👆の手順が完了したら、作成ボタンをクリックします。</List>
</Order>

:::caution

上記のプロジェクト設定は、Re:Earth 内部でのみ表示されます。プロジェクトを公開する際には、OGP や公開用のプロジェクト情報を別に設定することができます。公開用の設定については、[こちら](/user-manual/project-and-workspace/project/public-settings).をご確認ください。

:::
