For the purpose of editing and managing projects, Re:Earth has three main pages:
- [Dashboard](/user-manual/getting-started/understanding-reearth-ui/Dashboard)
- [Editor](/user-manual/getting-started/understanding-reearth-ui/editor)
- [Setting Page](/user-manual/getting-started/understanding-reearth-ui/setting-page)

[Header](/user-manual/getting-started/understanding-reearth-ui/header) is a common part of these three pages. The function of the Header will vary from page to page.
