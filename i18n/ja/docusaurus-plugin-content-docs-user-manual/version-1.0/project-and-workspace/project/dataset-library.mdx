インポートしたデータはすべてデータセットとして Re:Earth 内に保存されます。詳しくは[データセット]をご覧ください。

データセットライブラリは、以下の場所で見つけることができます：  
編集画面 -> ヘッダーメニュー -> データセット

<Image path={require("./img/5-2.png").default} />

データセットライブラリーでは、現在のプロジェクトにインポートした全てのデータセットを確認・管理することができます。

<Image path={require("./img/5-1.png").default} />

## データセットの追加と削除

<Image path={require("./img/5-3.png").default} />

<Order>
  <List>
    <strong>データセットを追加</strong>{" "}
    では、お手元のPCにあるデータをインポートすることができます。
  </List>
  <List>
    <strong>削除</strong>{" "}
    は、データセットカードの上にマウスを置くと表示され、データセットの削除を行うことができます。
  </List>
</Order>

現時点では、データセットの管理は、データセットの追加と削除のみとなっていますが、今後の開発で機能を拡張していく予定です。

:::caution
  データセットを削除する際は、そのデータセットがプロジェクトで使用されていないことを確認してください。データセットが接続されている場合、接続されているプロジェクトが欠損する可能性があります。
:::
