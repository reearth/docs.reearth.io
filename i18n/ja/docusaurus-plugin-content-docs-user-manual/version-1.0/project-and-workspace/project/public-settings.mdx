公開設定では、公開されているプロジェクトに関する情報を設定することが出来ます。

<Image path={require("./img/4-1.png").default} />

<Order>
  <List>
    <strong>公開ステータス</strong>{" "}
    現在のプロジェクトの公開状況を確認することができます。
  </List>
  <List>
    <strong>ベーシック認証</strong>{" "}
    プロジェクト閲覧時に、閲覧者に対して求めるユーザーネームとパスワードを設定することができます。詳細は[こちら]をご確認ください。
  </List>
  <List>
    <strong>公開詳細設定</strong>{" "}
    Re:Earth外部に公開されるプロジェクトの基本情報(OGPを含む)を設定することができます。詳細は[こちら]をご確認ください。
  </List>
  <List>
    <strong>サイト設定</strong>{" "}
    カスタムドメインを設定することが可能です。初期状態は、Re:Earthのサブドメインが使用されています。詳しくは[こちら]をご確認ください。
  </List>
</Order>
