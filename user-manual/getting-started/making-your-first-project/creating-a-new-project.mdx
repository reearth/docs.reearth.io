After logging in, the first page you will see is the [Dashboard](/user-manual/getting-started/understanding-reearth-ui/Dashboard). Here is where new projects are created.

Follow these steps to create a New Project:

<Image path={require('./img/2-1.png').default} />

<Image path={require('./img/2-2.png').default} />

<Image path={require('./img/2-3.png').default} />

<Order>
  <ListByNumber>
    Click the New project button to open the Initialize project dialog modal
  </ListByNumber>
  <ListByNumber>
    Fill in the project name
  </ListByNumber>
  <ListByNumber>
    Fill in the project description
  </ListByNumber>
  <ListByNumber>
    Select and add the Thumbnail image
  </ListByNumber>
  <ListByNumber>
    Confirm to create a new project
  </ListByNumber>
  <ListByNumber>
    Once completed, you will find a new project card appearing on the project list
  </ListByNumber>
</Order>

At this point you have created your first Re:Earth project. Clicking the Edit button on the project card will take us to the project we just created.
