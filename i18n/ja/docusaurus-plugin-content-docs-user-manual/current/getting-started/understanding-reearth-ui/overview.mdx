プロジェクトの編集や管理を行うために、Re:Earthには大きく分けて3つのページがあります。
- [ダッシュボード](/user-manual/getting-started/understanding-reearth-ui/Dashboard)
- [編集ページ](/user-manual/getting-started/understanding-reearth-ui/editor)
- [設定ページ](/user-manual/getting-started/understanding-reearth-ui/setting-page)

[ヘッダー](/user-manual/getting-started/understanding-reearth-ui/header) は、これら3つのページの共通部分です。ヘッダーの機能はページごとに異なります。
まずは、ヘッダーについて説明致します。
